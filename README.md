# 미달이 - 우리 학과 장학금 소식 한 눈에 챙겨보기

미달이가 물어다 주는 우리 학과 장학금 소식


## :pushpin: Features

- 단과 대학, 학과 별 장학금 소식 확인
- 학교 공지사항 장학금 소식 확인
- 원하는 장학금 이름 검색
- 튜토리얼
